#### Курс предназначен для изучения популярных NoSQL баз данных, которые потом можно будет интегрировать в свой архитектурный ландшафт
*  Доступен для заказа B2B клиентам. Можно расширить или исключить какую-либо часть, добавить лабораторные и домашние работы.

#### Для кого этот курс:
*  Разработчики: Кто хочет научиться работать с различными NOSQL решениями для создания современных приложений.
*  Администраторы баз данных: Кто занимается настройкой, оптимизацией и поддержкой NOSQL решений в production-средах.
*  Аналитики данных: выбор оптимального NOSQL решения для обработки и анализа больших объемов данных.
*  DevOps-инженеры: Кто развертывает и автоматизирует NOSQL решения.

#### Итоговые навыки:
*  Умение работать с различными NOSQL решениями на всех уровнях: от базовых операций до продвинутых функций
*  Навыки оптимизации производительности и масштабирования NOSQL продуктов
*  Понимание принципов кластеризации и репликации
*  Знание инструментов для мониторинга, бэкапов и администрирования основных NOSQL решений

#### Инвестируйте в «результат», а не «информацию»:
*  Уникальная экспертиза «из первых рук», собранная за 25+ лет работы с высоконагруженными системами. Это знание, которого нет в книгах и стандартных мануалах.
*  Это не теоретические вебинары. Каждое занятие — это продуктовая практика на тестовых стендах, максимально приближенных к боевым.
*  Сотрудники пройдут обучение на реальных кейсах, а не на синтетических примерах. Все методики уже доказали свою эффективность под высокой нагрузкой.
*  Контролируйте вовлеченность сотрудников: к лекциям по желанию идут домашние задания для закрепления материала или лабораторные онлайн занятия, но вы сами решаете, насколько глубоко погружаться.

01. NOSQL vs RDBMS. CAP теорема
02. MongoDB — архитектура и продвинутые возможности
03. Шардированный реплицированный кластер MongoDB
04. Couchbase
05. Cassandra
06. ClickHouse архитектура + engines
07. Шардированный реплицированный кластер ClickHouse
08. Redis
09. etcd
10. Kafka
11. KafkaRabbitMQ
12. KafkaPrometheus. Grafana
13. KafkaTarantool
14. KafkaManticore Search
15. KafkaCockroachDB
16. KafkaApache Superset
17. KafkaNDB Cluster
